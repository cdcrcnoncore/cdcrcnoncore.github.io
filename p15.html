How did you get to know about opportunity?

On campus


What is your Job Profile?

Data Science Intern


Can you give a general discription of the interview with some interview questions?

"1st Round Online (60 mins)
3 Probability Questions (Bayes Theorem) (Medium Difficulty) 4 Basic Python Questions (Lists and Dictionaries related) (Easy Difficulty) 3 Coding Questions (only python as the coding language) (Easy Difficulty) This round was piece of cake if you are familiar with python
2nd Round Telephonic interview (30 mins)
After introduction, they asked about machine learning basics (like supervised / unsupervised, K Means, Logistic Regression etc.). Then they asked python questions (what are generators, iterators, return vs yield, lambda function). There were also a few OOP related questions. I had mentioned economics course and also being a part of FnE club (fincom), in resume, so they asked about basic economics terminology like GDP and Macro vs Micro Economics etc. Equity vs Debt, and kind of debts."


How did you prepare for it?

I was already familiar with python, and I had revised some machine learning basics.


Any tips for juniors?

This was basically more of computer science profile and my experience in python helped me, so you might want to have strong hold on python and ML basics.
